Needle Engine viewer

stub · priority for visionOS

Scope

  • Engine: Needle Engine — Unity-authored, exports to WebXR-enabled web. Primary reason for selection: works in visionOS Safari WebXR-AR, the critical Apple Vision Pro hedge (spec 015).
  • Render target: visionOS Safari (primary), Quest browser (secondary), Android Chrome, desktop.
  • Cannot run in UAAL (Unity-as-library embedded). Web-only by design.
  • XRAI I/O: File API + fetch, shared with the main viewer.
  • Multiplayer: Needle ships networking samples; will adopt LiveKit for consistency.

Status

Not yet implemented. Activation trigger (per RUNTIMES_EVALUATION O1 visionOS column): Apple Vision Pro install base > 5% of XR users, or Apple unlocks WebXR-AR on iOS Safari. Spec 015 has the full trigger doc.

Why Needle

Unity-to-web export gives Portals' VFX authoring pipeline a web path without reimplementation. Critical for the "one XRAI doc, every engine" claim when visionOS becomes a real target.
