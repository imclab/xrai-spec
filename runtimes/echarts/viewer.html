⚡ ECharts-GL + WebGPU XRAI viewer

stub — v1.1 target

Scope

  • Graph layout: ECharts graphGL (Apache 2.0, ~300KB) + optional WebGPU compute shader for 10K+ node force simulation.
  • Render target: desktop browsers + mobile Safari/Chrome. Heavy-data view when force-graph budget exceeded.
  • XRAI I/O: same File API + fetch patterns as main index.html.
  • Multiplayer: LiveKit (spec 010).

Status

Not yet implemented. Stub referenced by main viewer's engine picker. Full impl lands with RFC TBD — see VIEWER_ARCHITECTURE.md § Layout layer.

When implemented: passes the same conformance harness as threejs/, so XRAI docs render identically modulo engine-specific shading.

Why this engine

ECharts-GL is proven at 10K+ node graphs with interactive filtering/search. Spec 006 (KB Visualizer) already selects ECharts-GL for the developer/analytics tier — this viewer reuses that decision.
