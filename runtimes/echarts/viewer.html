xrai. echarts-gl · webgpu

ECharts-GL + WebGPU

stub · v1.1 target

Scope

  • Graph layout: ECharts graphGL (Apache 2.0, ~300KB) + optional WebGPU compute shader for 10K+ node force simulation.
  • Render target: desktop browsers + mobile Safari/Chrome. Heavy-data view when force-graph budget is exceeded.
  • XRAI I/O: same File API + fetch patterns as the main viewer.
  • Multiplayer: LiveKit (spec 010).

Status

Not yet implemented. Stub is referenced by the main viewer's engine picker. Full impl lands with RFC TBD — see VIEWER_ARCHITECTURE.md § Layout layer.

When implemented: passes the same conformance harness as threejs/, so XRAI docs render identically modulo engine-specific shading.

Why this engine

ECharts-GL is proven at 10K+ node graphs with interactive filtering and search. Spec 006 (KB Visualizer) already selects ECharts-GL for the developer and analytics tier — this viewer reuses that decision.
