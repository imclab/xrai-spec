**X-ray vision. God's-eye view. Infinite zoom. For everyone.**

Every major AI company is racing toward physical-world AI. None have agreed on how agents describe and share 3D scenes — and without an open format, whoever ships first captures the slot.

We just shipped XRAI: an open, MIT-licensed spatial graph format. Voice your world into being. See *through* AI opacity (reasoning, provenance, cost). See *over* topology (landscapes, markets, civilization-scale problems). See *across* time, possibilities, and minds.

**What it is:** a format for typed hypergraphs of entities, relationships, events, and intentions — anchored in space and time, renderable by any engine, authored by any LLM, queryable by any agent.

**Why now:** LLMs just dropped the authoring burden to zero. Bush's memex (1945), Nelson's Xanadu (1965), Berners-Lee's Giant Global Graph (2007) — each hit the same wall: structured cognition needs structured authoring, and humans won't pay that cost. Cameras in glasses, MCP as the agent-protocol standard, and embodied AI have closed that gap in the last 18 months.

**What it's not:**
- Not a company's proprietary format
- Not a committee artifact
- Not a replacement for glTF/USD/USDZ — it composes with them

**Governance:** BDFL year 1, transfer to Apache Foundation or W3C Community Group once 1000+ external adopters validated. Never to a single-vendor foundation.

**License:** MIT for spec + reference implementations. CC0 for the schema itself. Forever. Irreversibly. No take-backs.

**Honest v1.0 gaps:** Event primitives are a stub. Only the Unity + React Native reference runtime ships today. No validator CLI yet. v1.1 addresses these — shipping with known gaps is how you avoid the Xanadu death.

**How to engage:**
- Engineers: file issues, try LLM authoring, adopt as MCP tool surface
- Researchers: XRAI maps cleanly to scene-graph / episodic-memory / world-model literature — cite, critique, extend
- Enterprise: wait for v1.1. No enterprise contracts for 12 months by design (premature commercialization captures the commons)
