Case study — Portals → XRAI

One ingested document, four viz modes. Same data, different lens. Demonstrates RFC 0009/0012 — M sources × N decoders collapses to M + N when a neutral semantic middle exists.

Source files: examples/15-portals-to-xrai.xrai.json, 16-youtube-video.xrai.json, 17-wikipedia-article.xrai.json. Adapter: js/adapters/index.js (13 adapters incl. new youtube). Decoders: web-force-graph, unity-vfx-hologram. Ontology + facets: ontology/v1.json, archetypes: archetypes/registry.json.
